# LightweightShopping
This is a Lighweight Shopping Web Application made
1) Backend: Python
2) Framework: Flask
3) DB: MongoDB
4) FrontEnd: HTML/CSS

Steps
1) Install all the dependencies from the requirements.txt file
2) Hit command python main.py
